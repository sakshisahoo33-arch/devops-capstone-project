# devops-capstone-project

This repository contains the Customer Accounts Microservice developed as part of the IBM DevOps Capstone Project.

## Project Overview
This project demonstrates Agile planning, Test-Driven Development (TDD), Continuous Integration (CI), Continuous Delivery (CD), Docker, Kubernetes, and OpenShift deployment for a Flask-based RESTful microservice.
